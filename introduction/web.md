### 网址组成

- protocol 协议

  - http://
  - https://
  - ftp://

- domain 域名

  - ipv4
  - www.xxx.com
  - 127.0.0.1
  - localhost

- port 端口

    :int(0~65535)

- path 路径

    /path1/path2

- query parameter 查询参数

    ?key1=value1&key2=value2

- anchor 锚点

    #anchor
